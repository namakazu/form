## Claude Codeが自動でやること

1. 関連ファイルを読む
2. コードを書く
3. 「編集しますか？」と確認してくる → **Enterで承認**

---

## 自分でやること（手動）

### GASに反映する
```bash
cd ~/form/gas
clasp push
```

### GASエディタで再デプロイ
1. GASエディタを開く
2. 「デプロイ」→「既存のデプロイを管理」
3. 「新しいバージョン」で再デプロイ
4. ※ WebhookURLは変わらないのでLINE側の設定変更は不要

### フロントエンドに反映する
```bash
cd ~/form
git add .
git commit -m "変更内容のメモ"
git push
```
→ Vercelが自動でデプロイしてくれる
